This is a page that we will scribble on.

Drawing is off by default and needs to be activated by slecting a pen color. Here are some commands:

  • ctrl-` (next to the 1) erase and turn off drawing
  • ctrl-1 Yellow Pen
  • ctrl-2 Green Pen
  • ctrl-3 Cyan Pen
  • ctrl-4 Red Pen
  • ctrl-5 White Pen
  • ctrl-6 Blue Pen
  • ctrl-7 Magenta Pen
  • ctrl-8 Black Pen
  • ctrl-- Make pen narrower
  • ctrl-= Make pen wider

Start drawing by selecting a color and then draw y holding down the mouse and moving. You can change the color or width of the pen using Control keys.
